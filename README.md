# DriftLog

> 목적 의식에서 오는 안도감

홍수로 가라앉은 한국. 물에 잠긴 도시들 사이를 배로 항해하며, 서비스를 켜놓고 일상을 즐기세요.
조작이나 점수가 아니라, 그냥 시간이 지나면 가고 싶은 곳으로 흘러가도록 설계했습니다.

## 이런 경험입니다

DriftLog는 빠르게 깨야 하는 게임이 아닙니다.
음악을 틀어두듯 켜놓고, 다른 일을 하다 가끔 들여다보는 항해 경험입니다.
배는 천천히 바다를 건너고, 그동안 하늘과 물빛이 바뀌고, 이따금 예상치 못한 일이 일어납니다.
화면을 계속 지켜보지 않아도 괜찮습니다.
흘러가는 것을 지켜보는 것 자체가 이 서비스의 전부입니다.

가라앉은 도시들 사이에는 먼저 떠난 가족이 남긴 흔적이 흩어져 있습니다.
목적지를 정하고 출항하는 것 말고는 해야 할 일이 거의 없습니다.
아무것도 안 하는 것이 개발의 의도입니다.
바쁜 하루 한켠에 작은 바다를 띄워두고, 어디론가 향하고 있다는 느낌을 주는 것이 서비스의 목적입니다.

<br>

## 어떻게 흘러가나

플레이어는 세 가지 상태를 오가며 항해합니다.

- **정박** — 한 도시에 머무르며 풍경과 흔적을 살핍니다
- **항해** — 다음 도시를 향해 바다를 건넙니다. 시간이 지나면 도착합니다
- **일시정지** — 항해를 잠시 멈추고 머뭅니다

항해 중에는 현실 날씨에 맞춘 날시가 바다 위에 펼쳐지고, 가끔 우연한 사건이 일어납니다.
모든 항해는 항해 일지에 자동으로 기록되며, 원한다면 직접 메모를 덧붙일 수 있습니다.
지나온 도시와 발견한 흔적은 그대로 쌓여, 내가 어디까지 흘러왔는지를 보여줍니다.

<br>

## 주요 기능

- 도시 간 항해와 시간 기반 도착 — 켜놓으면 알아서 흘러가는 흐름
- 도시별 흔적 발견 — 첫 방문에만 단서가 공개되는 연출
- 날씨 테마와 항해 중 무작위 이벤트
- 가족 재회 엔딩 —  모든 흔적 수집 + 최종 도착지 도달 시 발동
- 항해 일지 자동 기록 및 사용자 메모
- Google 소셜 로그인
